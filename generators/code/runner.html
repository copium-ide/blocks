Copium IDE Project Runner

This page uses the logic from code.js and assembler.js to run your project.

Instructions:
  1. Host your project.js and its required modules (e.g., variables.js) on a service with CORS enabled, like a GitHub Gist.
  2. Make sure the modules array in your project.js uses the full, raw URL of the module files.
  3. Paste the raw URL for your project.js below and click "Assemble".
